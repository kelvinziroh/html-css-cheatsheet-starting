HTML Multimedia cheatsheet

Multimedia Tags

Tag Name Description
<img> Image tag Allows you to embed an image on the page
<audio> Audio tag Allows you to embed an audio file on the page
<video> Video tag Allows you to embed a video file on the page

Multimedia Attributes

Attribute Name Description
src Source attribute Contains path to image, audio or video file you want to embed
alt Alternative text This is a textual description of an image
controls Controls attribute Provide a way for a user to control the audio or video
width Width attribute Specify the width of an embedded image or video on the page
height Height attribute Specify the height of an embedded image or video on the page
